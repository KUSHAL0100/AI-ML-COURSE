##### **Rule-Based System**

    * Uses fixed human-made rules.
    * Example: “If cholesterol > 240, mark as high risk.”
    * Works based on predefined conditions.
    * Does not learn from new data automatically.





##### **Data-Driven Approach**

    * Uses machine learning and patterns from data.
    * The system learns relationships automatically from past patient records.
    * Can improve accuracy when more data is provided.



Example:

Instead of using only cholesterol, a data-driven model may also learn that age, blood pressure, smoking, and exercise together affect heart disease risk.
